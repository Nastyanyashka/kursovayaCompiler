Постановка задачи

Ассоциативный массив — абстрактный тип данных, позволяющий хранить пары вида «ключ, значение».

Для описания ассоциативного массива в языке C# используется служебное слово Dictionary.

Формат записи:

"Dictionary<$type1, $type2> $name = new Dictionary<$type1, $type2>();"

Параметр $name передает название массива, параметр $type1 передает тип ключа, параметр $type2 передает тип значения. Тип ключа или значения может представлять int или string.

Примеры:

Тип ключа int, тип значения string:

Dictionary My_dict1 = new Dictionary();

Тип ключа int, тип значения int:

Dictionary dictionary123 = new Dictionary();

Тип ключа string, тип значения string:

Dictionary name = new Dictionary();
